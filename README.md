# bendavis-site

Personal website for **Ben Davis** — bennettsdavis.site. Next.js 14 (App Router) on Vercel.

- **Owner editing (no code):** log in at `/admin`, then use **Edit Mode** on the live site (click text, swap photos, drag to reorder, add blocks) and **Quick edits** at `/admin/quick` for email/links/promos.
- **Content** lives in `data/content.json`. **Design tokens** in `app/globals.css`.
- **AI assistants:** see [`AGENTS.md`](./AGENTS.md) for how to edit this project safely.
- **Owner's guide:** [`public/onboarding.html`](./public/onboarding.html) (live at `/onboarding.html`).

Push to `main` to deploy. Roll back anytime via Vercel → Deployments → `...` → Instant Rollback.
